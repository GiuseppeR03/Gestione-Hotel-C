# Gestione-Hotel-C
"Programma in C per la gestione di prenotazioni alberghiere tramite Tabelle Hash e Code"

Questo programma in C funziona come un sistema gestionale per una catena di alberghi. 
In parole povere, legge da dei file di testo quali sono gli hotel disponibili e chi ci vuole andare a dormire, 
smista le persone negli alberghi corretti e crea un sistema per ritrovare le prenotazioni all'istante.
